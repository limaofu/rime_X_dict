# rime_X_dict
rime extention dictionarys &amp; schema

# ① cof_punctuator
符号输入方案，输入日文时不用切换大小写字母，按读音输入，每输入一个音节时供选择的都是平假名在上，片假名在下，<br>
可按分号键选择第二个候选字上屏，跟五笔的习惯一样。
